# Spring Boot Users CRUD (REST API)

CRUD simple de usuarios con Spring Boot siguiendo buenas prácticas:
Controller + Service + Repository + DTO + Mapper + Exceptions.

## Tecnologías
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven

## Cómo ejecutar
### Requisitos
- Java instalado
- Maven (o usar el Maven de IntelliJ)

### Ejecutar
```bash
mvn clean install
mvn spring-boot:run
